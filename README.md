# Multishop Microservice

**Multishop Microservice** is a scalable e-commerce application built with **.NET Core** and **microservice architecture**.  
The project is designed to simulate a real-world e-commerce system, managing core business processes such as **product management, category management, orders, and user authentication**.  

This project is currently **35% complete**, with the **core modules and infrastructure implemented**.  
It also serves as a learning platform for **microservice patterns, containerization with Docker, and modern .NET development practices**.

---

## Technologies & Tools

Some technologies are actively used in the project, while others will be integrated in later stages.  

- **Backend & API:** ASP.NET Core, API Consume, Rapid API  
- **Databases:** MSSQL, PostgreSQL, MongoDB, SQLite  
- **ORM & Data Access:** Entity Framework Core, Dapper  
- **Architecture & Design Patterns:** Onion Architecture, CQRS, Mediator, Repository  
- **Authentication & Authorization:** IdentityServer, JWT, JWT Bearer  
- **Real-Time Communication:** SignalR  
- **Caching & Performance:** Redis  
- **Containerization & Deployment:** Docker  
- **API Management:** API Gateway, Ocelot Gateway  
- **Frontend / Communication:** Ajax, Swagger, Postman  
- **Others:** Google Drive Integrated Photo Upload

> ⚠️ Note: The project is under active development; some technologies and features have not been fully implemented yet.
